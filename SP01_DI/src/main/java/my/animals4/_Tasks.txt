1. Перепишите код Runner'a так, чтобы он вместо xml конфигурации контекста
использовал кодовую (класс конфигурации придется добавить, конечно).

по умолчанию, враг - собака; собаки бесстрашны. Но у отдельных особей может быть
и не по умолчанию. Кроме того, у животного должен быть метод Escape (удрать).
3. Сконфигурирйте наш контекст так, чтобы в нем были Шарик, Бобик и Мурка.
Бобик и Мурка имеют врагов по умолчанию, а Шарик труслив и боится кошек.
4. Добавьте бизнес-процесс Встреча (Rendevouz). Наши животные выходят на свидание,
"осматриваются" и, если видят врага среди участников процесса, удирают.

Весь программный код должен быть loosely coupled.
(За исключением, конечно, класса конфигурации, если он используется).
